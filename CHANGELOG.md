# Changelog

## 0.1.0 - 2026-03-07

初回の実用リリースです。小規模チーム向けのシンプルなガントアプリとして、日常運用に必要な機能を一通り揃えました。

- ガント画面でのタスク作成、削除、日付変更、進捗管理
- predecessor ベースの依存関係と依存違反の可視化
- 担当者割り当て、プロジェクトメンバー管理、管理画面
- 検索、担当者、状態、期間によるフィルタ
- 遅延タスクの可視化、今日へ移動、直前 Undo
- CSV / XLSX の取込と出力
- 変更履歴表示
- Docker / LAN 運用向けの構成と README 整備
- frontend / backend のテスト拡充と pre-commit hook 導入
